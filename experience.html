Clothing Detection
Python YOLOv12.s Roboflow Computer Vision Ultralytics
Deskripsi

Proyek ini merupakan sistem deteksi objek pakaian berbasis computer vision menggunakan model YOLOv12. Model dilatih dengan dataset kustom dari Roboflow untuk mengenali beberapa kategori pakaian pada gambar, yaitu dress, jacket, pants, shirt, short, skirt, sweater, T-shirt. Alur proyek mencakup persiapan dataset, training model, validasi performa menggunakan metrik object detection, serta pengujian prediksi pada gambar baru. Selain model utama, proyek ini juga dilengkapi dengan antarmuka interaktif berbasis Gradio sehingga pengguna dapat mengunggah gambar, mengatur confidence threshold, dan melihat hasil deteksi beserta detail objek yang teridentifikasi.

Fitur Utama
  • Melatih model YOLOv12 untuk mendeteksi beberapa kategori pakaian dari dataset kustom.
  • Menggunakan dataset hasil anotasi dari Roboflow dalam format YOLO.
  • Melakukan evaluasi performa model menggunakan metrik seperti precision, recall, mAP50, dan mAP50-95.
  • Menampilkan visualisasi hasil evaluasi dalam bentuk grafik metrik dan confusion matrix.
  • Melakukan prediksi pada gambar baru untuk melihat hasil deteksi objek pakaian secara langsung.
  • Menyediakan demo interaktif berbasis Gradio untuk upload gambar dan pengujian model secara sederhana.

Analisis Sentimen Isu Perundungan Mahasiswa dari Reddit
Python Sentiment Analysis Reddit API Text Preprocessing
Deskripsi

Proyek ini bertujuan untuk menganalisis sentimen publik dari diskusi Reddit terkait kasus perundungan mahasiswa dengan memanfaatkan pendekatan Natural Language Processing (NLP). Data dikumpulkan menggunakan Reddit API dari beberapa subreddit dan kata kunci yang relevan, kemudian diproses melalui tahapan text preprocessing seperti case folding, penghapusan URL dan karakter non-teks, normalisasi kata informal bahasa Indonesia, serta filtering stopwords.

Setelah tahap preprocessing, teks diterjemahkan ke bahasa Inggris untuk mendukung analisis tambahan, sementara analisis sentimen utama dilakukan menggunakan pendekatan lexicon-based sentiment analysis dengan kamus kata positif dan negatif yang disesuaikan dengan konteks data. Hasil akhir proyek ini berupa klasifikasi sentimen positif, negatif, dan netral, persentase distribusi sentimen, serta visualisasi word cloud untuk menampilkan kata-kata yang paling sering muncul dalam diskusi.

Fitur Utama
  • Mengambil data postingan Reddit dari beberapa subreddit berdasarkan keyword tertentu menggunakan Reddit API.
  • Melakukan preprocessing teks bahasa Indonesia, termasuk cleaning text, normalisasi kata tidak baku, dan penghapusan stopwords.
  • Menerjemahkan teks hasil preprocessing ke bahasa Inggris sebagai pendukung eksplorasi data.
  • Menghitung skor sentimen menggunakan pendekatan lexicon-based sentiment analysis.
  • Mengklasifikasikan opini ke dalam kategori sentimen positif, negatif, dan netral.
  • Menampilkan distribusi sentimen dalam bentuk grafik batang.
  • Membuat visualisasi word cloud untuk melihat kata-kata yang paling sering muncul pada data.

Credit Card Fraud Detection using Autoencoder
Python Machine Learning Anomaly Detection Autoencoder Fraud Detection
Deskripsi

Proyek ini membangun sistem deteksi fraud pada transaksi kartu kredit menggunakan pendekatan anomaly detection berbasis Autoencoder. Dataset diproses terlebih dahulu melalui tahap pembagian data, standardisasi fitur numerik, serta pemisahan data normal dan fraud untuk mendukung skenario pelatihan unsupervised. Model Autoencoder dilatih hanya menggunakan transaksi normal agar mampu mempelajari pola transaksi yang wajar dan menghasilkan reconstruction error yang lebih tinggi pada data anomali.

Setelah proses training, reconstruction error dihitung pada validation set dan test set. Threshold optimal ditentukan melalui threshold tuning berdasarkan F1-score terbaik pada validation set, sehingga model tidak hanya bergantung pada threshold tetap. Evaluasi akhir dilakukan pada test set menggunakan confusion matrix, classification report, ROC-AUC, serta visualisasi distribusi reconstruction error untuk membandingkan pola transaksi normal dan fraud. Proyek ini menekankan bagaimana Autoencoder dapat digunakan sebagai pendekatan anomaly detection untuk kasus fraud detection pada data yang tidak seimbang.

Fitur Utama
  • Melakukan preprocessing dan standardisasi data transaksi kartu kredit sebelum proses pelatihan model.
  • Melatih Autoencoder hanya pada transaksi normal untuk mempelajari pola data non-fraud dan mendeteksi anomali melalui reconstruction error.
  • Menentukan threshold terbaik menggunakan validation set dan F1-score agar prediksi fraud lebih adaptif.
  • Mengevaluasi performa model menggunakan confusion matrix, classification report, ROC-AUC, serta visualisasi distribusi reconstruction error.

Deskripsi

MYP (Make Your Plan) merupakan proyek perancangan antarmuka aplikasi mobile yang dibuat untuk membantu pengguna mengelola waktu dan aktivitas harian secara lebih terstruktur. Konsep aplikasi ini berfokus pada pengaturan jadwal tugas, penjadwalan acara, pengingat pribadi, serta fitur world clock untuk memantau waktu di berbagai lokasi. Selain menampilkan alur penggunaan mulai dari sign up, sign in, hingga halaman utama, proyek ini juga menekankan konsistensi visual, penyusunan layout mobile, dan perancangan pengalaman pengguna untuk aplikasi produktivitas.

Fitur Utama
  • Antarmuka mobile untuk pengelolaan jadwal tugas dan aktivitas harian.
  • Fitur penjadwalan acara dengan pengaturan tanggal, waktu, deskripsi, dan detail aktivitas.
  • Fitur alarm pribadi sebagai pengingat aktivitas penting pengguna.
  • Fitur world clock untuk melihat waktu di berbagai lokasi secara praktis.
